# 폭락장 대비 LOC 추가 매수 전략

> 일일 정액 투자 기반, 하락 시 자동으로 더 많이 사는 전략

---

## 핵심 개념

### 기본 아이디어

**일일 투자금을 고정**하고, 가격이 하락할수록 더 많은 수량을 매수합니다.

```
가격이 떨어지면 → 같은 돈으로 더 많이 산다 → 평단가 자연스럽게 하락
```

### 왜 이 전략인가?

| 일반적인 물타기 | 이 전략 |
|----------------|---------|
| 추가 자금 필요 | 일일 투자금 고정 |
| 감정적 판단 개입 | 미리 설정된 규칙 |
| 타이밍 고민 | LOC로 자동 체결 |

---

## 공식

### 기본 공식

```
추가 매수 가격 = 일일 투자금 ÷ (기본 수량 + N × step)
```

- **일일 투자금**: 매일 투자할 고정 금액
- **기본 수량**: 기본 가격에서 매수할 수량
- **N**: 추가 매수 레벨 (1, 2, 3, ...)
- **step**: 레벨당 증가 주식 수 (기본값 1)

### Step 파라미터

**step**은 각 레벨에서 몇 주씩 추가할지 결정합니다:

| step | 레벨 1 | 레벨 2 | 레벨 3 | 레벨 4 | 레벨 5 |
|------|--------|--------|--------|--------|--------|
| 1 | +1주 | +2주 | +3주 | +4주 | +5주 |
| 2 | +2주 | +4주 | +6주 | +8주 | +10주 |
| 3 | +3주 | +6주 | +9주 | +12주 | +15주 |

- **step=1**: 촘촘한 가격 간격, 소량씩 추가 매수 (기본)
- **step=2+**: 넓은 가격 간격, 더 많은 수량 추가 매수

### 공식 유도

일일 투자금이 고정이므로:

```
일일 투자금 = 기본 가격 × 기본 수량
            = 추가 매수 가격 × (기본 수량 + N × step)
```

정리하면:

```
추가 매수 가격 = (기본 가격 × 기본 수량) ÷ (기본 수량 + N × step)
```

### 하락률 공식

```
하락률(%) = (1 - 기본 수량 ÷ (기본 수량 + N × step)) × 100
```

## 체결 시나리오

### 종가별 체결 결과 (예시 1 기준)

| 종가 | 체결되는 주문 | 총 수량 | 총 투자금 |
|------|--------------|---------|----------|
| $10.50 | 없음 | 0주 | $0 |
| $10.00 | 기본만 | 10주 | $100 |
| $9.00 | 기본 + N1 | 11주 | $100 |
| $8.00 | 기본 + N1~N2 | 12주 | $100 |
| $7.00 | 기본 + N1~N4 | 14주 | $100 |
| $6.00 | 기본 + N1~N6 | 16주 | $100 |
| $5.00 | 기본 + N1~N10 | 20주 | $100 |

> **핵심**: 어떤 가격에 체결되든 총 투자금은 항상 $100!

### 시각화

**step=1 (기본)**
```
가격
$10.00 ─┬─ 기본 10주
        │
$9.09  ─┼─ +1주 (총 11주)
        │
$8.33  ─┼─ +1주 (총 12주)
        │
$7.69  ─┼─ +1주 (총 13주)
        │
$7.14  ─┼─ +1주 (총 14주)
        │
$6.67  ─┼─ +1주 (총 15주)
        │
  ↓     ↓
하락    추가 매수
```

**step=2 (넓은 간격)**
```
가격
$10.00 ─┬─ 기본 10주
        │
        │  (빈 구간)
$8.33  ─┼─ +2주 (총 12주)
        │
        │  (빈 구간)
$7.14  ─┼─ +2주 (총 14주)
        │
        │  (빈 구간)
$6.25  ─┼─ +2주 (총 16주)
        │
  ↓     ↓
하락    추가 매수
```

## 실전 적용

### Step 1: 기본값 설정

```
1. 일일 투자금 결정 (예: $100, $500, $1000)
2. 기본 매수 가격 결정 (현재가 또는 목표가)
3. 기본 수량 계산 = 일일 투자금 ÷ 기본 가격
```

### Step 2: 추가 매수 간격(step) 결정

```
step=1: 촘촘한 간격 (1주씩 추가) - 소폭 하락에도 대응
step=2: 넓은 간격 (2주씩 추가) - 큰 하락에 집중
step=3: 더 넓은 간격 (3주씩 추가) - 폭락에만 대응
```

### Step 3: 추가 매수 가격 계산

```
N=1: 일일 투자금 ÷ (기본 수량 + 1 × step)
N=2: 일일 투자금 ÷ (기본 수량 + 2 × step)
N=3: 일일 투자금 ÷ (기본 수량 + 3 × step)
...
```

### Step 4: LOC 주문 설정

```
LOC [기본 가격] × [기본 수량]
LOC [N1 가격] × [step]주
LOC [N2 가격] × [step]주
LOC [N3 가격] × [step]주
...
```

### Step 5: 최소 가격 설정 (선택)

폭락 한도를 정해서 그 이하는 주문하지 않음

```
예: 최소 가격 $5 설정 → $5 이하 추가 매수 없음
```

## 장점과 주의사항

### 장점

1. **리스크 관리**: 일일 투자금 고정으로 과도한 투자 방지
2. **자동화**: LOC 주문으로 감정 배제
3. **평단가 관리**: 하락 시 자연스럽게 평단가 하락
4. **단순함**: 한 번 계산하면 매일 같은 주문 반복

### 주의사항

1. **레버리지 ETF 주의**: TQQQ, SOXL 등은 변동성 큼
2. **무한 하락 대비**: 최소 가격 또는 최대 추가 매수 횟수 설정
3. **유동성**: 너무 낮은 가격은 체결 안 될 수 있음
4. **정기 점검**: 평단가 변동 시 재계산 필요
